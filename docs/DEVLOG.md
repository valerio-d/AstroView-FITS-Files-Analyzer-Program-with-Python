## 2. Research: Understanding FITS Files and Main Libraries
**Flexible Image Transport System** (FITS) **files** can stro images, spectra, data cubes, and medadata. FITS Files are made of **Header Data Units**, HDUs. The primary HDU, HDU0, is the main image. Anything after that is extra images, tables, sprectra, etc.

**Astropy** is a Python library used to work with **Astronomy** in Python, offering many tools and packages for astronomy. 
**NumPy** is a Python library used for working with arrays.
**Matplotlib** is a Python library that allows users to plot graphs using data.
