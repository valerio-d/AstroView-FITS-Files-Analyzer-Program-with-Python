# AstroView-FITS-Files-Analyzer-Program-with-Python
##About the project
A lightweight Python program that uses Astropy, Numpy, and Matplotlib for FITS Files Visualization and Analysis.
Astronomical data comes in `.fits` format, which isn't accessible to standard image viewers.
I developed this project to combine astrophysics and programming.

## Tech Stack:
Python 3.10, Astropy, NumPy, Matplotlib
